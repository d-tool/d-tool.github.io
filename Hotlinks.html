HOTLINKS

Was sind Hotlinks

Zum Beispiel die Möglichkeit mit der Maus im Channel auf einen HTTP-Link zu klicken, damit sich der Browser öffnet. Textfunktionen die man auf gepostete Inhalte von Chatfenstern ausführen kann

LM = linke Maustaste | RM = rechte Maustaste


Alle Channel und @Windows (beim Mausklick auf ein Releasenamen)

LM + STRG
Es wird gefragt ob das Release auf allen Sites gesucht werden soll
(zB. auch in @d-tool_trigger, recht hilfreich da man bei Triggern meistens auch Interesse hat es zu laden)
LM + SHIFT
Es wird gefragt ob das Release als Request gesucht werden soll, vor allem interessant bei Request auflistungen um nochmal per Hand zu checken


d-tool Fenster

Mausklick auf ein Suchergebnis

STRG + LM
Es wird gefragt ob das Suchergebnis in den Downloadordner geladen werden soll. Wenn für FTPRush noch kein Downloadordner angegeben wurde wird auch danach gefragt, bei cbftp wird der Standard Download-Ordner von cbftp genommen
SHIFT + LM
Es wird gefragt ob dahin connected werden soll. Der Pfad wird auch in die Zwischenablage kopiert (nur FTPRush)

Mausklick auf einen Sitenamen bei Ausgabe von /dtinfo

LM
Es werden die Target- und Sourcesites von cbftp angezeigt, die man per /dcbsite erhält
RM + STRG
Einstellen der Targetsites
RM + SHIFT
Einstellen der Sourcesites
LM + STRG
Logincheck für diese Site ausführen

Mausklick auf den Status "off!" beim Logincheck bei Ausgabe von /dtinfo

LM
Logintimer starten

Mausklick auf einen Pfad bei Ausgabe von sitesetup>sections>section management>show sections

LM
Der Sektionspfad wird in die Zwischenablage kopiert, damit man ihn in cbftp pasten kann wenn man die Sektionen einstellt
