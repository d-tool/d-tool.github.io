Hotlinks

Was sind Hotlinks?
Zum Beispiel die Möglichkeit mit der Maus in mIRC auf einen HTTP-Link zu klicken ist ein Hotlink. Also Textfunktionen die man auf gepostete Inhalte von Chatfenstern ausführen kann. Folgende Funktionen gibt es:

Im d-tool Fenster beim klick auf ein Suchergebnis:
STRG+LinkeMaustaste: Es wird gefragt ob das Suchergebnis in den Downloadordner geladen werden soll. Wenn noch kein Downloadordner angegeben wurde wird auch danach gefragt
SHIFT+LinkeMaustaste: Es wird gefragt ob dahin connected werden soll. Der Pfad wird auch in die Zwischenablage kopiert.

in allen Channels, die als Racechans im d-tool angegeben sind beim klick auf ein Release:
STRG+LinkeMaustaste: Es wird gefragt ob das Release auf allen Sites gesucht werden soll
SHIFT+LinkeMaustaste: Es wird gefragt ob das Release als Request gesucht werden soll, vor allem interessant bei Request auflistungen um nochmal per Hand zu checken.
