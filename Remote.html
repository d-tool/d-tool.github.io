REMOTE

Sollte man den at auf einem Server bzw. einer VM laufen lassen, kann man sich zb. dtinfo oder Rush-Stats remote anzeigen lassen.
Dazu dann die dtoolremote-at.mrc in das mIRC laden, mit dem auch der at läuft. Die Befehle stehen im Rechtsklickmenü unter Remote im d-tool.
Wenn man d-tool Zuhause gar nicht geladen hat, kann man sich die d-tool-remotemenü.mrc reinladen, um nur die Befehle für Remote zu bekommen
